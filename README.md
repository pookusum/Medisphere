## 🚀 Getting Started

### Prerequisites
- Node.js v16+
- Google Maps API key
- WebSocket server

### Installation
```bash
# Clone repository
git clone https://github.com/yourusername/hospital-finder.git

# Install dependencies
cd hospital-finder
npm install

# Set environment variables
cp .env.example .env
